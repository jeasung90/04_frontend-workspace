텍스트 관련 스타일 (text~)

color : 텍스트 색상 지정

        선택자{
            color : 색상명 | 16진수 | RGB(X, X, X) | RGBA(X, X, X, 투명도) 
            | HSL(X, X, X) 채도명도, HSLA(X, X, X, 투명도)
        }

text-decoration : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

        선택자{
            text-decoration : nonn | underline | overline | line-through;
        }

text-transform : 영문 텍스트의 대소문자 변환시 사용

        선택자 {
            text-transform : uppercase | lowercase | capitalize
        }
  1. uppercase : 모든 영문자를 다 대문자로
  2. lowercase : 모든 영문자를 다 소문자로
  3. capitalize : 영문자의 첫번째 글자를 대문자로

text-shadow : 택스트에 그림자 효과를 줄 때 사용

        선택자 {
            text-shadow : 가로거리(x) 세로거리(y) [번짐정도] [색상]
        }
