font-size : 글꼴의 크기를 변경할 때 사용

선택자{font-size : 크기(px, em(배수), %)}
    dddddd
  • 고정크기 px테스트
  • 가변크기 em테스트
  • 가변크기 %테스트

font-weight : 글꼴의 굵기를 표현할 때 사용

        선택자{
            font-weght : nomal | bold | boder | lighter | 100 | 400 | 900
        }
  • 원래 굵기
  • 굵은 굴끌로 변경
  • 원래 굵기보다 굵게
  • 원래 굵기보다 얇게

font-variant : 영문 텍스트 문구를 작은 대문자로 변결할 때 사용

        선택자 {font-variant : small-caps;}

font-style : 텍스트 문구를 기울이가조 할 때 사용

선택자 {font-style : normal | italic | oblique;}

italic : 기울임 글꼴

oblique : 원래 글자를 기울인셈
