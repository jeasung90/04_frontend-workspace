form 태그 내에 submit 버튼 클릭시 해당 form 안에 작성된
사용자가 입력한 값들을 서버로 넘기면서 요청하는 역활을 수행

form action="" method=""

* action 속성 : 해당 form에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로를 제시
* method 속성 : 요청 전송 방식을 지정하는 속성 (get(기본값) / post)
> get 방식 :요청시 사용자가 입력한 값들이 url에 노출되는 방식
> post 방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식
form 태그 내에 submit 버튼 클릭스 action 지정한 서버로 사용자가 입력한 값들이
넘어가면서 요청됨 (사용자가 입력한 값들은 항상 key = value 세트로 전달)

라디오버튼 (type="radio")

-제시한 여러개의 값들 중에서 오로지 한개만 선택해야 될 때
(name 속석값이 똑같은 것들끼리 하나의 그룹으로 지어짐)
- 제출시 현재 선택된(checked) 라디오 버튼의 value 값이 넘어감
(해당 라디오 버튼 선택시 어떤 값을 넘길껀지 value 값을 명시해줘야함!!)

성별 :

체크박스 (type ="checkbox")

- 제시한 여러개의 값들 중에서 여러개 선택가늘할 경우
- 제출시 현재 선택된(checked) 체크박스의 value값 넘어감(넘기고자 하는 value값 명시해둬야 한다!)

select 태그와 option 태그

- 사용자에게 직접 키보드로 값을 입력받는게 아니라 내가 제시한 여러개 옵션(목록)들 중에서 선택할 수있게끔
- 제출시 현재 선택된(selected) option 값이 넘어감
- 각각의 option에 value 값을 명시하지 않았을 경우 contnet 영역의 값이 넘어감 - value 값을 명시했을 경우 value값이 넘어감
